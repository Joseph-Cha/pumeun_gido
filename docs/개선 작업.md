먼저 현재 저장된 메모리를 확인하고 ADHD 서비스 관련 정보를 제거하겠습니다.현재 저장된 메모리가 없는 상태입니다. 혹시 다른 대화에서 저장된 정보가 있다면 해당 대화에서 확인이 필요할 수 있습니다.

품은기도 서비스에 대해 완전히 새로운 관점에서 개선 기획안을 다시 작성해드리겠습니다.

# 품은기도 서비스 개선 기획안

## 핵심 인사이트: "관리 도구"에서 "기도 습관 동반자"로

현재 MVP는 기도 제목을 **저장하고 분류하는 도구**입니다. 하지만 사용자가 진정 원하는 것은 **꾸준히 기도하는 사람이 되는 것**입니다. 이 간극을 메우는 방향으로 개선해야 합니다.

---

## 1단계: 즉시 적용 가능한 개선 (1-2주)

### 1.1 앱 진입 경험 재설계

**현재 문제**: 홈 화면이 기도 목록 나열 → "창고"처럼 느껴짐

│  ┌─────────────────────────┐   │

│  │ "취업 면접이 잘 되게"    │   │
│  │ 7일째 함께 기도 중        │   │
│  │              [기도했어요] │   │
│  └─────────────────────────┘   │
│                                 │
│  ┌─────────────────────────┐   │
│  │ 엄마                     │   │
│  │ "건강검진 결과가 좋게"   │   │

│  │              [기도했어요] │   │

│                                 │
│  오늘 2개의 기도가 기다리고 있어요│

**핵심 변화**:
- 전체 목록이 아닌 "오늘 기도할 것"에 집중
- 며칠째 기도 중인지 표시 → 지속성 인식
- [기도했어요] 버튼으로 즉각적 행동 유도

## 2단계: 습관 형성 시스템 (2-4주)

### 2.1 기도 알림 (가장 중요한 기능)

**왜 중요한가**: 기도 앱의 핵심 가치는 "잊지 않고 기도하게 해주는 것"

| 알림 유형 | 시점 | 메시지 예시 |
|-----------|------|-------------|
| 일일 기도 | 설정 시간 | "민준이와 수진이가 기도를 기다려요" |
| 신규 등록 | D+1 | "어제 등록한 기도, 오늘도 함께해요" |
| 장기 기도 | 7일마다 | "민준이를 위한 기도 7일째, 계속 함께해요" |
| 재방문 유도 | 3일 미접속 | "품은 기도들이 기다리고 있어요" |

**알림 설정 화면**:
```
┌─────────────────────────────────┐
│  기도 알림 설정                  │
├─────────────────────────────────┤
│                                 │
│  매일 기도 알림                  │
│  [ON ●━━━━━━━━━━━━━━━━━━━━━━]   │
│                                 │
│  알림 시간                       │
│  [오전 6:30 ▼]                  │
│                                 │
│  알림 메시지 미리보기            │
│  ┌─────────────────────────┐   │
│  │ 🙏 품은기도               │   │
│  │ 민준이와 수진이가         │   │
│  │ 기도를 기다려요           │   │
│  └─────────────────────────┘   │
│                                 │
└─────────────────────────────────┘
```

### 3.3 기도 나눔 (성장의 핵심)

**시나리오**:
1. 수진이 기도 제목을 카카오톡으로 공유
2. 받는 사람이 링크 클릭 → 앱 설치/실행
3. "수진이가 기도를 부탁했어요" 메시지와 함께 기도 제목 자동 등록
4. 수진에게 "3명이 함께 기도 중" 알림

**공유 메시지 예시**:
```
🙏 기도 부탁드려요

"취업 면접이 잘 되게 기도해주세요"

함께 기도해주실 분은 아래 링크를 눌러주세요
[품은기도에서 함께 기도하기]
```

## 핵심 성공 요소

**1. 습관 형성이 핵심입니다**
- 알림 없이는 앱이 잊혀집니다
- "오늘의 기도"로 매일 돌아올 이유를 만드세요

**2. 감정적 보상을 설계하세요**
- 응답의 기쁨, 연속 기도의 성취감
- 차가운 관리 도구가 아닌 따뜻한 동반자

**3. 셀 리더를 핵심 사용자로 삼으세요**
- 1명의 셀 리더 = 8명의 잠재 사용자
- 그룹 기능은 성장의 레버
