```
품은기도는 지인들의 기도 제목을 체계적으로 관리하고 기도 응답을 추적할 수 있는 기도 동반자 앱입니다.

셀 모임, 소그룹, 개인 대화에서 전달받은 기도 제목을 흩어지지 않게 한 곳에서 관리하세요.

📖 주요 기능

• 기도 제목 관리
  - 기도 제목을 쉽게 등록하고 관리할 수 있어요
  - 기도 중 / 응답받음 상태를 변경하며 기도의 여정을 기록해요
  - 메모 기능으로 기도하며 느낀 점을 남길 수 있어요

• 카테고리 분류
  - 일반, 건강, 직장, 가정, 관계, 신앙 등 카테고리별로 분류해요
  - 필터 기능으로 원하는 기도 제목만 모아 볼 수 있어요

• 중보 대상 관리
  - 기도 요청자별로 기도 제목을 정리해요
  - 누구의 기도 제목인지 한눈에 확인할 수 있어요

• 간편 로그인
  - Google 계정으로 간편하게 시작해요
  - iOS에서는 Apple 로그인도 지원해요
  - 여러 기기에서 데이터가 동기화돼요

🙏 이런 분께 추천해요

• 셀/소그룹 리더로 여러 셀원의 기도 제목을 관리하시는 분
• 지인들의 기도 제목을 마음에 품고 꾸준히 기도하시는 분
• 기도 응답의 기록을 남기고 싶으신 분
• 분산된 기도 제목을 한 곳에서 관리하고 싶으신 분

💝 품은기도와 함께

기도 제목을 카톡이나 메모장에 흩어두지 마세요.
품은기도에서 소중한 기도 제목을 체계적으로 관리하고,
응답받은 기도를 기록하며 감사의 여정을 이어가세요.
```

## 7. 타겟층 및 콘텐츠

### 타겟 연령층
```
Q: 이 앱의 타겟 연령층은 무엇인가요?

✅ 13세 이상만 선택
```

> **13세 이상만 선택하는 이유:**
> - 어린이 대상 앱이 아님 (COPPA 규정 회피)
> - 종교/기도 앱으로 성인 사용자 타겟
> - 어린이 타겟 시 추가 심사 및 규정 적용됨

### 어린이 관심 유도 여부
```
Q: 이 앱이 어린이에게 의도치 않게 관심을 끌 수 있나요?

✅ 아니요
```

> **이유:** 만화 캐릭터, 게임 요소 없음. 밝은 색상의 유아용 디자인 아님. 종교적 콘텐츠로 어린이 관심 대상 아님.

### 광고 포함 여부
```
Q: 앱에 광고가 포함되어 있나요?

✅ 예 (AdMob 배너/전면 광고 사용)
```

> 광고는 AdMob을 통해 게재되며 어린이 대상이 아닌 일반 광고입니다.

### ⚠️ 주의사항
어린이 타겟 선택 시 문제점:

## 8. 앱 액세스 권한

Play Console → **앱 콘텐츠** → **앱 액세스 권한**

### 설정 방법
```
✅ "모든 기능 또는 일부 기능이 제한됨" 선택
```

> 품은기도는 Google/Apple 로그인이 필수이므로 테스트 계정 정보를 제공해야 합니다.

### 테스트 계정 정보 (예시)
| 항목 | 값 |
|------|-----|
| 이름 | 테스트 계정 |
| 사용자 이름/이메일 | (테스트용 Gmail 주소) |
| 비밀번호 | (테스트 계정 비밀번호) |
| 기타 안내 | Google 로그인 버튼을 눌러 이 계정으로 로그인하세요. |

### 테스트 계정 준비 체크리스트
- [ ] 테스트용 Gmail 계정 생성
- [ ] 앱에 로그인하여 샘플 데이터 입력
  - 기도 요청 2-3개 (기도 중 상태)
  - 응답된 기도 1-2개 (응답받음 상태)
  - 기도 요청자 2-3명
- [ ] 계정 정보를 Play Console에 정확히 입력
- [ ] 2단계 인증 비활성화 (심사자 접근 용이)

### 주의사항
- 테스트 계정은 **실제로 로그인 가능**해야 함
- 비밀번호가 **심사 기간 동안 유효**해야 함
- 계정이 **잠기거나 차단되지 않도록** 관리
