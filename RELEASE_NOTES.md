## v1.1.0 - iOS (2025-12-17)

### AdMob 광고 시스템 적용

iOS 앱에 AdMob 광고 시스템이 적용되었습니다.

#### 변경 사항

- **배너 광고**
  - 설정 화면 하단에 배너 광고 표시

- **전면 광고**
  - 기도 제목 저장 시 5회마다 전면 광고 표시

- **ATT (App Tracking Transparency)**
  - iOS 14.5+ 사용자에게 추적 권한 요청 다이얼로그 표시
  - 사용자 동의 후 맞춤형 광고 제공

- **app-ads.txt 설정**
  - AdMob 광고 검증을 위한 app-ads.txt 파일 호스팅

#### 플랫폼별 빌드 정보

| 플랫폼 | 버전 | 빌드 번호 |
|--------|------|-----------|
| iOS | 1.1.0 | 3 |

## 버전 관리 규칙

### 버전 형식

`{major}.{minor}.{patch}+{buildNumber}`

- **major**: 주요 기능 변경 또는 호환성이 깨지는 변경
- **minor**: 새로운 기능 추가 (하위 호환성 유지)
- **patch**: 버그 수정 및 작은 개선
- **buildNumber**: 플랫폼별 빌드 번호 (자동 증가)

### 버전 파일

플랫폼별 버전은 `version.json` 파일에서 관리됩니다:

```json
{
  "android": {
    "versionName": "1.0.0",
    "buildNumber": 2
  },
  "ios": {
    "versionName": "1.0.0",
    "buildNumber": 1
  }
}
```

빌드 스크립트 실행 시 해당 플랫폼의 빌드 번호가 자동으로 증가합니다.
